IntCatch Vision Data Set

This data set has been developed in the context of WP4 "Autonomous and radio controlled boats for water quality monitoring" that aims at developing autonomous boat platforms for the INTCATCH water monitoring methodology, so that non-technical users can operate the system.

Data contained in the IntCatch Vision Data Set are provided without any warranty about their usability.
